# AI Email Automation System

A robust backend pipeline that parses, classifies, and routes emails automatically using a simulated AI-ready logic. Built to demonstrate enterprise-level automation workflows.

## 🔄 Automation Workflow

- **Email classification**: Categorizes incoming emails (e.g., Order Inquiry, Complaint, Support Request).
- **Intelligent routing**: Automatically routes emails to the appropriate department (Sales, Support, Tech).
- **JSON data storage**: Structurally saves all processed data for audit and reporting.
- **Simulated system actions**: Mimics real-world outcomes like sending notifications or triggering workflows.

## 🚀 Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the automation:
   ```bash
   python src/main.py
   ```

## 🛠 Tech Stack
- Python
- OpenAI API (Simulated/Ready)
- python-dotenv
